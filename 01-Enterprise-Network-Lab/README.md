# 01 - Enterprise Network Security Lab

## Objective
Build a realistic enterprise-style network security lab for learning and portfolio projects.

## Lab Components
- Windows Server (Domain Controller)
- Windows 10 Client
- Kali Linux Attacker VM
- pfSense Firewall
- Splunk SIEM

## Goals
- Network segmentation
- Log monitoring
- Attack simulation and detection
